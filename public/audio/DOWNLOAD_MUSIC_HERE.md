# IMPORTANT: Download Background Music

The audio file needs to be downloaded manually.

## Quick Download Instructions:

### Option 1: Pixabay (Recommended - No Attribution Required)
1. Visit: https://pixabay.com/music/search/romantic%20instrumental/
2. Search for: "romantic piano" or "lo-fi love"
3. Preview and select a track you like
4. Click the green "Free Download" button
5. Save the file as `background-music.mp3` in this folder

### Option 2: Free Music Archive
1. Visit: https://freemusicarchive.org/search?quicksearch=romantic%20instrumental
2. Find a track with Creative Commons license
3. Download as MP3
4. Save as `background-music.mp3` in this folder

### Recommended Tracks:
- "Sentimental Piano" - soft and romantic
- "Acoustic Love Theme" - gentle and warm
- "Romantic Jazz Loop" - elegant and smooth

### Requirements:
- **Format**: MP3
- **File Name**: `background-music.mp3` (exactly)
- **Size**: Under 5MB (2-3MB ideal)
- **Length**: 2-5 minutes (will loop automatically)

## After Adding Music:

1. Refresh your browser (Ctrl+F5 or Cmd+Shift+R)
2. Click "Begin Our Story" button
3. Music should start playing automatically
4. Use the mute button (top-right) to control audio

## Alternative: Use Your Own Music

If you have a special song you want to use:
1. Convert it to MP3 format (use online converter if needed)
2. Trim it to 2-5 minutes for faster loading
3. Save as `background-music.mp3` in this folder

**Note**: Make sure you have the right to use any music you choose!
